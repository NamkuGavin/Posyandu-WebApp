# WHO Child Growth Standards data

The tab-separated reference tables in this directory are sourced from the
World Health Organization `anthro` package:

- Repository: https://github.com/WorldHealthOrganization/anthro
- Source directory:
  https://github.com/WorldHealthOrganization/anthro/tree/master/data-raw/growthstandards

Files used by this project:

- `lenanthro.txt`: length/height-for-age
- `weianthro.txt`: weight-for-age
- `wflanthro.txt`: weight-for-length
- `wfhanthro.txt`: weight-for-height
- `bmianthro.txt`: BMI-for-age

The application uses the WHO LMS formula and biological plausibility flag
ranges. Indonesian category labels follow Permenkes No. 2 Tahun 2020 on Child
Anthropometry Standards.
